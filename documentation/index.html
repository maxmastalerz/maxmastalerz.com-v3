Setup Development Environment

To kickstart the development of the web app with Rewy, at first need to setup the Gatsby & React development environment.

Following tools are needed to setup a react dev environment:

  • Node JS

    • NodeJS works as a node package manager behind the scene. It is recommended to download and install the latest version of Node JS from its official site http://nodejs.org/

Preparation

Step 1: You'll need to install Node.js >=v10.0+ (Recommended Version) (NPM comes along with it) and serve (a NPM package)

  • Projects Structure

    • In the downloaded folder you will found 2 projects on is

    • gatsby/without-strapi

      • This one is totally React based static template

    and the second one is
    • gatsby/with-strapi-cms

      • And this one with Strapi API support, and configured with Strapi.

      How to start Strapi API?

      Go to dexia-api root and open the terminal/git bash install all dependencies by npm install then npm run develop when project run the url will open automatically on your default browser and go to this link http://localhost:1337/admin/auth/login and login via this credentials,
      Email: hello@hibootstrap.com and the Password: HiBootstrap10 and it is the admin, you can learn more about Strapi here

      You will find the example code of Strapi/Graphql /src/components/Index

      And the GraphQL playground is http://localhost:8000/___graphql

      Now continue step 2

Step 2: Install the Gatsby CLI.

npm install -g gatsby-cli

Use command line tool i.e. powershell or terminal and navigate to the project(Rewy) root. This is powershell in windows and terminal on mac/linux.

Step 3: Go to the Rewy root folder and open the terminal and run npm install.

Step 4:

  • Start development server
  • gatsby develop
  • Gatsby will start a hot-reloading development environment accessible by default at http://localhost:8000.
  • Try editing the home page in src/pages/index.js. Saved changes will live reload in the browser.
  • Create a production build
  • gatsby build
  • Gatsby will perform an optimized production build for your site, generating static HTML and per-route JavaScript code bundles.
  • Serve the production build locally
  • gatsby serve
  • Gatsby starts a local HTML server for testing your built site.
